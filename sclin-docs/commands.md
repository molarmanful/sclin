## CMD: [``` type ```](#cmd-type)

Stack: ``` a -> STR ```

Type of `a`.


## CMD: [``` form ```](#cmd-form)

Stack: ``` a -> STR ```

`a` as formatted string.


## CMD: [``` >Q ```](#cmd-q)

Stack: ``` a -> STR ```

Converts `a` to `SEQ`.


## CMD: [``` >A ```](#cmd-a)

## CMD: [``` >M ```](#cmd-m)

## CMD: [``` >N ```](#cmd-n)

## CMD: [``` >F ```](#cmd-f)

## CMD: [``` >E ```](#cmd-e)

## CMD: [``` @$ ```](#cmd--9)

Stack: ``` (a >STR) -> ```

Loads ID `a` into local scope.
```sclin
"outer"=$a ( \a @$ a ) # $a
#a "inner"
```


## CMD: [``` @$$ ```](#cmd--10)

Stack: ``` (a >STR) -> ```

Loads ID `a` into global scope.
```sclin
\a @$$ ( "inner" =$a $a ) # a
#a "outer"
```

## CMD: [``` dip ```](#cmd-dip)

Stack: ``` a* b (f >FN) -> _* b ```

[``` pop ```](#cmd-pop)s `b`, [``` # ```](#cmd--12)s `f`, and pushes `b`.


## CMD: [``` \ ```](#cmd--11)

## CMD: [``` # ```](#cmd--12)

Stack: ``` a* f -> _* ```

```sclin

```


## CMD: [``` Q ```](#cmd-q-1)

Stack: ``` f' -> _' ```

```sclin
1 2 ( dups 3+` ) Q
```

## CMD: [``` &# ```](#cmd--18)

Stack: ``` a* (b >TF) f -> _* ```

[``` # ```](#cmd--12)s `f` if `b` is truthy.


## CMD: [``` |# ```](#cmd--19)

Stack: ``` a* (b >TF) f -> _* ```

[``` # ```](#cmd--12)s `f` if `b` is falsy.


## CMD: [``` ?# ```](#cmd--20)

Stack: ``` a* (b >TF) f g -> _* ```

[``` # ```](#cmd--12)s `f` if `b` is truthy; else [``` # ```](#cmd--12)s `g`.


## CMD: [``` ??# ```](#cmd--21)

Stack: ``` a* (b >MAP) -> _* ```

Iterates through each key-value pair of `b`.
For each pair: if the [``` Q ```](#cmd-q-1) of the key is truthy,
then [``` # ```](#cmd--12)s the value and short-circuits.


## CMD: [``` *# ```](#cmd--22)

Stack: ``` a* f (n >NUM) -> _* ```

[``` # ```](#cmd--12)s `f` `n` times.


## CMD: [``` !# ```](#cmd--23)

Stack: ``` a* f g -> _* ```

Tries to [``` # ```](#cmd--12) `f`; on error, pushes caught `ERR` and [``` # ```](#cmd--12)s `g`.


## CMD: [``` !Q ```](#cmd-q-2)

Stack: ``` f' -> TRY' ```

[``` Q ```](#cmd-q-1)s `f` and wraps the result in a `TRY`.


## CMD: [``` ~Q ```](#cmd-q-3)

Stack: ``` f' -> TASK' ```

[``` Q ```](#cmd-q-1)s `f` asynchronously, returning a future.


## CMD: [``` >! ```](#cmd--24)

Stack: ``` (e ERR) -> ```

Throws `e`.


## CMD: [``` ' ```](#cmd--25)

Stack: ``` (a >ARR) f -> ARR ```

[``` # ```](#cmd--12)s `f` on `a` as if it were a stack.
```sclin
[1 2 3 4] ( 5 swap ) '
```


## CMD: [``` '_ ```](#cmd-_)

Stack: ``` (a* >ARR) f -> _* ```

[``` # ```](#cmd--12)s `f` on the stack as if it were an `ARR`.
```sclin
1 2 3 4 1.+.map '_
```

Stack: ``` (a >NUM)' -> NUM' ```

## CMD: [``` |^ ```](#cmd--27)

Stack: ``` (a >NUM)' -> NUM' ```

Rounds `a` towards ∞.


## CMD: [``` X>b ```](#cmd-xb)

Stack: ``` (a >NUM)' (b >NUM)' -> ARR[NUM*]' ```

```sclin

```
```sclin

```


## CMD: [``` b>X ```](#cmd-bx)

Stack: ``` (a >ARR[>NUM*]) (b >NUM)' -> NUM' ```

## CMD: [``` ~>> ```](#cmd--99)

## CMD: [``` ~< ```](#cmd--100)

Stack: ``` (a >TASK)' -> TASK' ```

Ensures that `a` runs on a separate thread.


## CMD: [``` ~: ```](#cmd--101)

Stack: ``` (a >TASK)' -> TASK' ```

Ensures that `a` is memoized such that subsequent runs of the task return the same value.


## CMD: [``` ~:& ```](#cmd--102)

Stack: ``` (a >TASK)' -> TASK' ```

[``` ~: ```](#cmd--101) but only if `a` completes successfully.


## CMD: [``` ~$_ ```](#cmd-_-8)

## CMD: [``` ~% ```](#cmd--103)

Ensures that `a` will error if not completed within `n` milliseconds.


## CMD: [``` sleep ```](#cmd-sleep)
